# Registers

In a real Computer,
<br>
Registers are the fastest memory
<br>
Some examples of Registers are eax(x86), ebx(x86), edx(x86), and more
<br><br>
In BVM:
<br>
Registers are not really fast because <b>"They are Virtual"</b>
<br>
<br><br>
With Registers, You can store value in it
<br><br>
